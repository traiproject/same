# same — Brand (v0)

## Brand words

- **Reliable** — deterministic, consistent
- **Simple** — minimal, uncluttered
- **Fast** — snappy, modern

## Colors

- **Iris (Primary):** `#8B5CF6`
- **Ink (Text):** `#0B0F19`
- **Slate (Muted text/borders):** `#667085`
- **Mist (Background):** `#F6F7FB`
- **White:** `#FFFFFF`
- **Green (Success):** `#22A06B`
- **Red (Error):** `#D93025`

> Keep the palette tight: Iris for emphasis (links/CTAs/highlights), neutrals
> for everything else.

## Typography

- **UI / Headings:** Inter
- **Code / Mono:** GitHub Monaspace

Suggested CSS fallbacks:

- Inter:
  `Inter, ui-sans-serif, system-ui, -apple-system, Segoe UI, Roboto, Helvetica, Arial, "Apple Color Emoji", "Segoe UI Emoji"`
- Monaspace:
  `Monaspace, ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, "Liberation Mono", "Courier New", monospace`

## Logo + icon usage

- **Primary mark:** the bold `=` icon (use alone for favicons / app icons /
  tight spaces).
- **Logo file:** `same_logo.svg`.
- **Do**
  - Keep aspect ratio (scale uniformly).
  - Use generous clear space: **at least 1× the icon stroke height** on all
    sides.
  - Prefer Iris on light backgrounds; prefer white on Iris/dark backgrounds.
- **Don’t**
  - Don’t stretch, skew, or add effects (shadows, outlines, gradients).
  - Don’t place on busy imagery; use a solid Mist/Ink/Iris panel behind it.
